Инструкция по заполнению:
Условия задач описываем в текстовом файле.
Для описания формул используем LaTeX нотацию.

При создании файлов с заданием необходимо придерживаться структуры книги (см. README.md)
Каталоги (директории) необходимо именовать согласно структуру на английском языке.
Если создаете пустую директорию, необходимо выполнить файл git_cloud.py, который автоматический создает файлы README.md.
Пустые каталоги не обрабатываются git
